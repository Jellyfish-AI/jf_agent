Step 3: Ensure proper network configuration

On this page, you will find instructions on how to make sure your network configuration is set up properly.

Details

This step helps make sure your network is set up to work with the Jellyfish Agent. This is the final step in the agent setup guide!

Instructions

The agent will need to make various requests to function. Thus, we ask that you please ensure that your network firewall/proxies are configured such that the agent is able to:

  1. Make GET requests to the Jellyfish API at https://app.jellyfish.co:443/
  2. Make GET requests to your Jira and Git host(s) on port 443
  3. Make POST requests to URLs under s3.amazonaws.com on port 443
